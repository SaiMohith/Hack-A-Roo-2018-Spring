Analysis of Opioid Overdose in the New York State Area

Opioid overdose is a big problem in the United States. Here we have tried to gather data from different datasets and apply different methods to pre-process the data, visualize it and then decide a model and train it to get predictive outcomes.

We have used the following tools for our analysis:

  1. Pre-Processing: Microsoft Excel
  2. Scripting Language: R Script/R Studio
  3. Visualization: Tableau Desktop
  4. Data Mining: Twitter API, Google Maps API
  5. Web Page: HTML 5.0, Javascript, CSS

Where the Data is Collected From

Data for all the information visualizations used for the Analytics page is collected from the below two articles:
1. https://bchi.bigcitieshealth.org/cities/79/17612
2. https://www.cdc.gov/nchs/nvss/vsrr/drug-overdose-data.htm

We collected the data, pre-processed it and sorted into different csv files for our own analysis.
